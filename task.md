Full Stack MERN Learning Management System

Description

EduFlow is a full-stack Learning Management System (LMS) built using the MERN stack.


The platform allows instructors to create and manage courses, students to enroll and learn, and admins to oversee the entire system.
The goal of this project is to design and develop a real-world, scalable web application that demonstrates:
Secure authentication & authorization
Role-based access control
RESTful API development
Database design using MongoDB
Modern React frontend architecture


This project simulates how popular platforms like Udemy, Coursera, or Skillshare work at a foundational level.

User Roles
Admin
Has complete control over the system
Manages users, courses, and instructors
Monitors platform activity
Instructor
Creates and manages courses
Uploads lessons and course content
Tracks enrolled students
Student
Browses and enrolls in courses
Consumes learning content
Tracks course progress

Features
Authentication & Authorization Module
Features
User registration and login
JWT-based authentication
Password hashing using bcrypt
Role-based access control (Admin / Instructor / Student)
Protected routes in frontend and backend	
Outcome
Secure access to the application
Only authorized users can access specific features


Instructor Module
Features
Create, update, and delete courses
Upload course thumbnail images
Add lessons (video or text-based)
Publish or unpublish courses
View enrolled students
Outcome
Instructors can fully manage their learning content


 Student Module
Features
Browse and search available courses
View course details
Enroll in courses
Watch lessons
Track learning progress (percentage)
Leave reviews and ratings
Outcome
Students experience a complete learning workflow


Admin Module
Features
View all users and courses
Block or unblock users
Approve instructor accounts
Remove inappropriate courses
Dashboard with analytics:
Total users
Total courses
Enrollments
Outcome
Admin ensures platform quality and control


Dashboard & Analytics Module
Features
Role-specific dashboards
Course progress visualization
Enrollment statistics
Instructor performance overview


File Upload Module
Features
Upload course thumbnails
Upload lesson videos
Cloud storage using Cloudinary or similar

Database Design Overview
Users – stores all users with roles
Courses – course details and instructor info
Lessons – lesson-level content for each course
Enrollments – student-course mapping and progress
Reviews – ratings and feedback
